# SoulSpeak Features

## Core Features

### 1. Journaling
- Rich text journal entries
- Mood tagging per entry
- Tags and categorization
- Search across entries
- Favorites system
- Optional encryption per entry
- Writing prompts

### 2. Mood Tracking
- Quick mood logging with intensity scale (1-10)
- Trigger identification
- Energy level tracking
- Sleep quality correlation
- Visual trend charts
- Historical mood analysis

### 3. Affirmations
- Daily affirmation delivery
- Category browsing
- Custom affirmation creation
- Favorites collection
- Swipeable card interface

### 4. Breathwork
- Multiple techniques (Box, 4-7-8, Energizing)
- Animated breathing guide
- Session tracking
- Pre/post mood comparison

### 5. Gratitude
- Daily gratitude journaling (3 items)
- Reflection notes
- Streak tracking
- Historical review

### 6. Rituals
- Custom ritual creation
- Step-by-step guides
- Completion tracking
- Reminder scheduling

### 7. Growth Dashboard
- Milestone tracking with progress
- Achievement system
- Streak visualization
- Statistics overview

### 8. Community
- Anonymous sharing option
- Category-based posts
- Support and celebration
- Content moderation

### 9. Crisis Support
- Local crisis language detection
- Immediate hotline display
- Coping strategies
- Personal emergency contacts
- 988 Lifeline integration

### 10. Security
- Biometric lock
- AES-256 encryption
- Privacy overlay
- On-device only storage
